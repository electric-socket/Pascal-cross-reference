Summary of Pascal Cross-Reference

  1. Get either a command file or the main program
  2. If command file, process arguments
    1. Arguments are same as program's $CREF directive
  3. if main program, read and parse
    1. uses statement will give us names of units to look up
    2. if unit is not found, we go on without it
    3. cross-reference arguments may be given as {$CREF /(*$CREF followed by arguments
    4. Arguments may be given in a configuration file
    5. Certain keywords are default defined by the program. These cannot be overridden
      1. All standard pascal operators
        • AND
        • OR
        • NOT
        • NOR
      2. procedural type declarations
        • procedure
        • function
      3. block declarations
        • Statement block BEGIN END
        • Statement block CASE END
        • Statement block REPEAT  UNTIL
        • Record block RECORD END
        • Object block TBD
        • Ignore block ASM END
        • Other types may be declared, e.g. AAEC's LOOP construct
      4. Regular statements
        • FOR DO
        • IF THEN
        • WHILE DO
        • WITH DO
      5. Predefined types
        • CHAR
        • INTEGER
        • REAL
        • ARRAY
        • FILE
        • User may add other types
      6. Syntactic sugar
        • IN
        • OF
        • TO
        • DOWNTO
    6. Some identifiers only apply in certain contexts. Those are modifiers. Used outside of the context they are legal identifiers
      1. CONST modifiers - only in const declaration
      2. TYPE modifiers - only in type declaration
      3. VAR modifiers - only in var declaration
        • ABSOLUTE
        • REGISTER
      4. Procedural modifiers - only when declaring or defining a procedure or function
        • FORWARD
        • EXTERNAL
        • NAME
        • STDCALL
        • CDECL
        • PASCAL
    7. State setting or changing
      • INTERFACE - first time a procedural type is defined, it is automatically forward
      • all declarations are UNIT scope, public
      • IMPLEMENTATION - no longer auto forward
      • all declarations are  UNIT scope, private
      • Procedural - all declarations are private
    8. Declared identifiers
